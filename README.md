# ÉCLAT DE JARDIN — Site Premium

Site web ultra-premium pour Éclat de Jardin, spécialiste en piscines, aménagements extérieurs et containers architecturaux à Strasbourg.

## Technologie

- **HTML5** : Structure sémantique moderne
- **CSS3** : Glassmorphism, animations, responsive design
- **Vanilla JavaScript** : Interactions fluides
- **GSAP 3.12** : Animations avancées et scroll triggers
- **ScrollTrigger** : Animations au scroll parallax

## Fonctionnalités

✅ **Design glassmorphique premium** avec effets de verre  
✅ **Animations GSAP** sophistiquées  
✅ **Parallax scroll** sur images  
✅ **Header sticky** avec effet glassmorphique  
✅ **Chatbot avancé** avec connaissances techniques  
✅ **Google Maps** intégré  
✅ **Responsive** mobile → 4K  
✅ **Smooth scroll** natif

## Design System

### Couleurs

- **Primary**: #B3883E (Or métallique)
- **Secondary**: #003566 (Bleu profond)
- **Accent**: #F8E9C8 (Beige chaud)
- **Anthracite**: #33383B

### Typographie

- **Titres**: Cormorant Garamond (serif élégante)
- **Corps**: Inter (sans-serif moderne)

### Glassmorphism

```css
background: rgba(255, 255, 255, 0.08);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.15);
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.12);
```
